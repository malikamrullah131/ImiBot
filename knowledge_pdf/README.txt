FOLDER DOKUMEN REGULASI (KNOWLEDGE BASE PDF)
============================================

Letakkan file PDF peraturan/regulasi resmi Anda di folder ini. 
Sistem ImmiCare v4.0 akan otomatis memproses file-file ini untuk:

1. Ekstraksi Pengetahuan: Bot akan membaca pasal-pasal di dalamnya.
2. Vector Search (RAG): Pencarian jawaban akan merujuk langsung ke dokumen asli.
3. Audit AI: Admin bisa melakukan !audit untuk melihat kesesuaian jawaban AI dengan PDF di sini.
